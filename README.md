# University Event Management App - Project Plan

## Overview
This project presents a complete project plan for developing a university event managemnt mobile application.

## Objectives
- Improve communication between students and staff.
- Increase event participation.
- Reduce manual administrative work.

## Features
- Event listing and discovery
- Event registration system
- Push notification
- Feedback collection
- Admin dashboard

## Project Phases
1. Initiation
2. Planning
3. Execution
4. Monitoring and Controlling
5. Closing

## Methodology
Agile methodology with sprint-based developmentwas to ensure flexibility and continous improvement.

## Project Details
- Duration: 5 months
- Budget: €20,000
